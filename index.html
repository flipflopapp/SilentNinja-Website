Intelligent Mode

In Intelligent mode, SilentNinja looks up the history of an anonymous caller in call log to decide if the call should be blocked. The decision to block or not depends on parameters such as, are we getting repeated calls from that number, or did we dial the number earlier, or if we had a previous conversation. If no exception is found about an anonymous call, it is blocked.

The Blacklist is a powerful and robust Blacklist.

In addition to Intelligent mode, it supports blacklisting phone numbers.SilentNinja keeps the user in control and informed on how the algorithm behaves. Use SilentNinja in a meeting or when watching a movie or whenever you need to filter calls on your phone. Keep SilentNinja in a quickly accessible location on your phone.
